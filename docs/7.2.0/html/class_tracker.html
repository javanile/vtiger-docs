Detailed Description

This class is used to track the recently viewed items on a per user basis. It is intended to be called by each module when rendering the detail form. Portions created by SugarCRM are Copyright (C) SugarCRM, Inc. All Rights Reserved. Contributor(s): ______________________________________..
