◆ setDate()

setDate (   $year = null,
  $month = null,
  $day = null,
  $rollover = false 
)
protected

Set the date of this class The date defaults to now. If any part of the date is missing, it will default to whatever "now"'s date portion is. For instance, if the year provided is 2006 and no other portion is given, it will default to today's month and day, but in the year 2006. If, for any reason the date defaults to a nonsensical date, an exception will be thrown. For instance, if you specify the year as 2006, and the current date is february 29th, an exception will be thrown because the 29th of February does not exist in 2006.

Parameters
intThe year of this date
intThe month of this date
intThe day of this date
Exceptions
