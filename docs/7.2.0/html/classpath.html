static normalize (   $path)
static

Normalize the given path. On Windows servers backslash will be replaced with slash. Remobes unnecessary doble slashes and double dots. Removes last slash if it exists. Examples: path::normalize("C:\\any\\path\\") returns "C:/any/path" path::normalize("/your/path/..//home/") returns "/your/home"
