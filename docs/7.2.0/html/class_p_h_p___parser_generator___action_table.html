◆ acttab_insert()

acttab_insert ( )

Add the transaction set built up with prior calls to acttab_action() into the current action table. Then reset the transaction set back to an empty set in preparation for a new round of acttab_action() calls.

Return the offset into the action table of the new transaction.

Returns
int Return the offset that should be added to the lookahead in order to get the index into $yy_action of the action. This will be used in generation of $yy_ofst tables (reduce and shift)
