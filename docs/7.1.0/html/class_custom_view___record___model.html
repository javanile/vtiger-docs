transformToNewAdvancedFilter ( )

Function used to transform the older filter condition to suit newer filters. The newer filters have only two groups one with ALL(AND) condition between each filter and other with ANY(OR) condition, this functions tranforms the older filter with 'AND' condition between filters of a group and will be placed under match ALL conditions group and the rest of it will be placed under match Any group.
